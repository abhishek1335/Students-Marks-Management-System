ğŸ“– Usage Guide
For Faculty (Admin)
Navigate to the Login page.

Enter the Admin credentials (username/password).

Go to the Upload section.

Select the JNTUK Result PDF from your local machine and click upload.

The system will parse the file and confirm once the data is stored in the database.

For Students
Navigate to the Student Portal (Home page).

Enter your Roll Number (e.g., 19BQ1A05XX).

Click Get Results.

View your marksheet, SGPA, CGPA, and backlog status in a tabulated format.
